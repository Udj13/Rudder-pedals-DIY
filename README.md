# Rudder pedals DIY

Pedals made of scraps and sticks.

## Components:

Real pedals from an old An-2 aircraft

Digispark development board ATTINY85 (https://aliexpress.ru/item/2043055746.html)

Neodymium magnet (just magnetized to the piece of iron)

Hall sensor 49e
